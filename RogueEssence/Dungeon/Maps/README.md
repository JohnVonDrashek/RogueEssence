# Maps

Dungeon map data structures and management. Contains the Map class representing a single dungeon floor with all tiles, items, characters, and visibility state.

## Key Files

| File | Description |
|------|-------------|
| `Map.cs` | Main dungeon map class with tiles, entities, and game state |
| `BaseMap.cs` | Base class for maps with common tile functionality |
| `MapLayer.cs` | Visual layer for map decoration tiles |
| `MapItem.cs` | Item placed on the map floor |
| `MapStatus.cs` | Active map-wide status effect (weather, etc.) |
| `IMobSpawnMap.cs` | Interface for maps that spawn enemies |
| `IMapLayer.cs` | Interface for map layers |

## Relationships

- Generated by **LevelGen/** procedural generation
- Contains **Characters/** instances for all entities
- Uses **Tiles/** for terrain and effects
- Rendered by **DungeonScene** using **Content/**
- Loaded/saved by **DataManager**

## Usage

```csharp
// Access map tiles
Tile tile = map.GetTile(loc);

// Check walkability
bool canWalk = map.TileBlocked(loc, mobility);

// Spawn an enemy
Team team = map.MapTeams[teamIndex];
map.RespawnMob();

// Get all visible tiles for a character
map.GetCharSight(character);
```
