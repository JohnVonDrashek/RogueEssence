# Properties

Assembly metadata and project properties.

## Key Files

| File | Description |
|------|-------------|
| `AssemblyInfo.cs` | Assembly information including version, copyright, and CLR settings |

## Relationships

- Standard .NET project properties folder
- Version information used by **Versioning.cs**
- Compiler settings for the assembly

## Usage

This is a standard .NET Properties folder containing assembly metadata. The AssemblyInfo.cs file is automatically referenced by the compiler during build.
